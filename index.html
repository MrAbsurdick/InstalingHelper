Instaling Helper

Instaling

1. Tampermonkey

Install the tampermonkey extension in your browser.

2. Script

Copy the code block below and paste it into the tampermonkey extension's workspace.


// ==UserScript==
// @name         Instaling
// @namespace    http://tampermonkey.net/
// @version      1.0.0
// @description  Instaling auto complite
// @author       MrAbsurd
// @include      https://instaling.pl/ling2/html_app/app.php*
// @icon         https://www.google.com/s2/favicons?sz=64&domain=instaling.pl
// @grant        GM_getValue
// @grant        GM_setValue
// @require      https://raw.githubusercontent.com/MrAbsurdick/InstalingHelper/refs/heads/main/instaling.user.js
// ==/UserScript==

3. Ready!
